CacheManager

This C# module can be added to any ASP.Net project and allows browsing and pruning of the HttpCache

The project was inspired by Steve Smith's CacheManager.axd (released way back in 2006 on the AspAlliance site).

This re-implementation uses JSON serialization instead of XML serialization to show the values store and implements an opinionated view of how cache keys are constructed to allow a hierarchical view of the cache's current contents.
